# 🤖 hire-me-pls
Your AI job hunt wingman: finds the right jobs, filters the nonsense, and writes applications that actually makes sense.

# 💀 Why this exists? 
Applying to jobs is broken
- Scroll endlessly
- Apply to everything
- Get ghosted anyways
- Rethink your career choices

So instead of applying with all the noise around... 
**this will help you apply smarter**

# 🚀 What it should do?
- Find jobs from multiple sources
- Score them based on your profile
- Generate tailored resumes for each role
- Write custom cover letters that screams *please hire me* in a sophisticated manner
- **Shows only high quality matches**
